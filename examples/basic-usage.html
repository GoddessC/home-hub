1. Basic Clock (Minute-Only)

Default configuration with minute-only updates for optimal performance.

const clock = new AnalogClock(document.getElementById('basic-clock'));

2. Clock with Seconds

Enable seconds hand for dedicated clock screens.

const clock = new AnalogClock(document.getElementById('seconds-clock'), {
  showSeconds: true
});

3. Custom Size and Colors

Customize size, colors, and accessibility options.

const clock = new AnalogClock(document.getElementById('custom-clock'), {
  size: 200,
  accent: '#3b82f6',
  face: '#f8fafc',
  ariaLabel: 'Custom blue clock'
});

4. Dark Theme

const clock = new AnalogClock(document.getElementById('dark-clock'), {
  accent: '#fbbf24',
  face: '#1f2937',
  size: 120
});

6. Dynamic Updates

Update clock options dynamically without re-rendering.

7. Auto-Initialize with Data Attributes

Use data attributes for automatic initialization.

<div data-analog-clock 
     data-size="140" 
     data-accent="#10b981" 
     data-show-seconds="true"></div>

// Auto-initialize all clocks
AnalogClock.autoInit();
